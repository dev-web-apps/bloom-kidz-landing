**18.16.1**

## Run Locally

Firebase deployed url

```bash
  https://kidz-loom.web.app
```

Clone the project

```bash
  git clone https://github.com/dev-web-apps/kidz-bloom-web.git
```

Go to the project directory

```bash
  cd kidz-bloom-web
```

Install dependencies

```bash
npm i
```

Run development server

```bash
npm run dev
```

Create production build

```bash
npm run build
```
